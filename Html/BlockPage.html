Mastering GRANT & REVOKE in Oracle Database

by Sandhiya Priya | 3w | 829 views

Introduction

Oracle's Data Control Language (DCL) commands help regulate user access and protect database security.

  • GRANT: Assign privileges to users/roles.
  • REVOKE: Remove granted privileges.

What Can You Control?

  • Tables
  • Views
  • Stored Procedures
  • Other DB Objects

GRANT Examples

GRANT SELECT ON employees TO hr_user;

REVOKE Examples

REVOKE SELECT ON employees FROM hr_user;
