HTML Anchor Tag (<a>)

The HTML Anchor Tag (<a>) is used to create hyperlinks that link one page to another or to specific sections within a page. It is one of the most commonly used tags in HTML for navigation purposes.

Basic Syntax

<a href="URL">Link Text</a>

In this syntax, the href attribute specifies the destination URL of the link, and "Link Text" is the clickable text that users see.

Attributes of Anchor Tag

  • href: Specifies the URL of the page the link goes to.
  • target: Specifies where to open the linked document (e.g., _blank for a new tab).
  • title: Provides additional information about the link, often displayed as a tooltip.
  • rel: Specifies the relationship between the current document and the linked document (e.g., nofollow, noopener).

Conclusion

The anchor tag is essential for creating links in web pages, enabling easy navigation and connectivity between different resources on the internet.
