HTML Tags Demo

<html> Tags

The <html> tag represents the root of an HTML document. It is the container for all other HTML elements (except for the <!DOCTYPE> declaration).

<head> Tags

The <head> tag contains meta-information about the HTML document, such as its title, character set, styles, and links to external resources.

<title> Tags

The <title> tag defines the title of the HTML document, which is displayed in the browser's title bar or tab.

<body> Tags

The <body> tag contains all the visible content of the HTML document, such as text, images, links, and other media.
