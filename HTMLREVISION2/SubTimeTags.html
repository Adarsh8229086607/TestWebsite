HTML <sub> and <time> Tags

<sub> - Subscript Text

The <sub> tag is used to define subscript text, which appears slightly below the normal line of text. It is commonly used in chemical formulas and mathematical expressions.

Example: H2O represents water, where the "2" is a subscript indicating two hydrogen atoms.

<time> - Date and Time

The <time> tag is used to represent a specific time or date. It can include attributes like datetime to provide a machine-readable format.

Example: The event is scheduled for June 15, 2025.

<u> - Underlined Text

The <u> tag is used to underline text. It is often used to indicate misspelled words or for stylistic purposes.

Example: This is an underlined word.
