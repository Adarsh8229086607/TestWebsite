Input Type Password:
The <input type="password"> element is used to create a password input field in an HTML form. The characters entered in this field are masked (typically displayed as asterisks or dots) to enhance security and privacy.

Input Type Date:
The <input type="date"> element is used to create a date picker input field in an HTML form. It allows users to select a date from a calendar interface, ensuring that the input is in a valid date format.
Select:
The <select> element is used to create a drop-down list in an HTML form. It allows users to choose one or more options from a predefined list of choices. It have a multipleproperty to select multiple options, and also have a size attributes to define number of visible options.
